## ERRORES FRECUENTES

Para permitir una conexión fluida y natural es necesario crear puntos de anclajes a la misma altura en letras con diseño semejante para que la terminación de una letra se superponga con la siguiente, esto funciona para la mayoría de letras, en ocasiones debido a coordenadas mal tipeadas se pueden superponer letras o verse a difenetes alturas. Para letras con un diseño variable como la segunda letra en minúscula (b) es necesario rediseñar un glifo compuesto por las letras conflictivas y agregar instrucciones para que se activen según el orden que se generen, actualmente faltan diseños para b, v, w, y la mayoría de letras mayúsculas, los cuales se estarán añadiendo. Si encuentra alguno de esos errores puede enviar un mensaje al correo janomasterone@live.com indicando el error con una captura de pantalla o describiendo el error en texto escribiendo la palabra correctamente, enumerando cada letra iniciando desde la izquierda y luego especifica los numeros en los que se produce el error e indicando si el error es de continuidad o superposición para corregir a la brevedad posible. 

## RECOMENDACIONES DE SEGURIDAD

Para extremar precauciones, se recomienda iniciar desde una sesión live, descargar el archivo .ttf, analizarlo con algún antivirus o herramienta que busque irregularidades como OpenType Sanitizer (OTS), luego si todo va bien, puede proceder a agregar en los directorios de su sistema.

## SOBREESCRITURA

Para actualizar la fuente, se debe seguir el mismo procedimiento de la instalación, puede sbreescribir directamente el archivo.ttf o eliminar el antiguo y pegar el nuevo.

## CAMBIO DE NOMBRE

Para modificar el nombre de la fuente, debe modificar la metadata del archivo, lo cual es posible con la aplicación FontForge desde: Elemento > Nombres PS , luego cambiar nombre de la fuente, de la familia, genérico y derechos de autor, se recomienda hacerlo antes de instalar por primera vez si prefiere un nombre personalizado. Cambiar solo el nombre desde el gestor de archivos no realiza cambios en el nombre de la fuente para el sistema o las aplicaciones.
